# Lumina Lux

**Lumina Lux** is a sophisticated, high-end e-commerce concept and landing page designed for luxury architectural lighting and bespoke chandeliers. It features a modern, dark-themed aesthetic with elegant interactions and responsive layout.

## 🌟 Features

- **Immersive Hero Section:** An engaging first impression with an interactive, animated lighting toggle effect that transitions the ambiance from light to dark.
- **Premium UI/UX:** A carefully crafted design system featuring high-contrast typography, subtle gold accents, and fluid animations for a luxurious feel.
- **Responsive Layout:** fully optimized for desktop, tablet, and mobile devices to ensure a seamless shopping experience anywhere.
- **Modern Interactions:** Smooth scroll animations and element transitions powered by Framer Motion.

## 🛠️ Tech Stack & Skills

- **Frontend Framework:** React (v19)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion (`motion/react`)
- **Routing:** React Router v7
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd <your-repository-directory>
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
